## 🎯 Overview

Mathematics and Statistics are the core building blocks of Data Analytics. Every analytical technique—from descriptive reporting to predictive modeling—relies on mathematical reasoning and statistical principles.

This repository documents my learning journey through essential mathematical and statistical concepts required for becoming a Data Analyst. Each day contains structured notes, examples, practice problems, and real-world applications.

---

## 🚀 Learning Objectives

By completing this repository, I will be able to:

✔ Understand mathematical concepts behind data analysis

✔ Interpret statistical results with confidence

✔ Apply probability concepts to business problems

✔ Perform hypothesis testing and statistical analysis

✔ Understand regression techniques used in analytics

✔ Build a strong foundation for Machine Learning
