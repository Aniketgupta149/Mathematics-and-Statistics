# Day 02 Quick Notes

## Function

A relationship between input and output.

f(x) = y

---

## Domain

All possible input values.

---

## Range

All possible output values.

---

## Linear Function

f(x) = mx + b

m = slope

b = intercept

---

## Slope Formula

(y₂ − y₁)/(x₂ − x₁)

---

## Graph Components

* X-axis
* Y-axis
* Origin (0,0)

---

## Types of Functions

1. Linear
2. Quadratic
3. Exponential
4. Logarithmic

---

## Data Analytics Use Cases

* Sales Forecasting

* Trend Analysis
* Machine Learning Models
