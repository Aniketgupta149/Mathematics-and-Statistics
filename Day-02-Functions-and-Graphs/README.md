# What is a Function?

A function is a rule that maps every input to exactly one output.

Mathematically:

f(x) = y

Where:

* x = Input
* y = Output
* f = Function

Example:

f(x) = 2x + 3

If x = 4

f(4) = 2(4) + 3

f(4) = 11

Input = 4

Output = 11

---

# Real-Life Example

Imagine a taxi service charges:

Base Fare = ₹50

₹10 per kilometer

Function:

Fare = 50 + 10x

Where:

x = Distance traveled

If distance = 5 km

Fare = 50 + 10(5)

Fare = ₹100

# Domain and Range

## Domain

The set of all possible input values.

Example:

f(x) = x²

Domain:

All Real Numbers

(-∞, ∞)

---

## Range

The set of all possible output values.

For:

f(x) = x²

Range:

[0, ∞)

Because squares cannot be negative.

---

# Types of Functions

## 1. Linear Function

Form:

f(x) = mx + b

Where:

* m = slope
* b = y-intercept

Example:

f(x) = 3x + 2

Characteristics:

* Straight-line graph
* Constant rate of change

---

## 2. Quadratic Function

Form:

f(x) = ax² + bx + c

Example:

f(x) = x²

Characteristics:

* U-shaped graph
* Used in optimization problems

---

## 3. Exponential Function

Form:

f(x) = aˣ

Example:

f(x) = 2ˣ

Applications:

* Population growth
* Compound interest
* User growth

---

## 4. Logarithmic Function

Form:

f(x) = log(x)

Applications:

* Data transformation
* Handling skewed data
* Machine Learning preprocessing

# Slope

Slope measures how steep a line is.

Formula:

Slope = (y₂ − y₁) / (x₂ − x₁)

Example:

Points:

(1, 3)

(2, 5)

Slope = (5 − 3)/(2 − 1)

Slope = 2

Meaning:

For every 1 unit increase in x, y increases by 2.

---

# Increasing Function

As x increases, y increases.

Example:

f(x) = x

Positive slope

---

# Decreasing Function

As x increases, y decreases.

Example:

f(x) = -x

Negative slope

---

# Constant Function

Output remains unchanged.

Example:
