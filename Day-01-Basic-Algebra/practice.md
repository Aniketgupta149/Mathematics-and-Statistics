# Practice Questions

1. Solve:

        x + 8 = 15          . Ans : x = 15 - 8; x = 7.

2. Solve:

         2x = 20             . Ans : x =   20/2, x = 10.

3. Solve:

       3x + 5 = 20       . Ans : 3x = 20 - 5,  3x = 15,  x = 15/3 , x = 5.

4. Calculate:

       (4 + 6) × 3         .Ans : (10) * 3 ;   30.

5. Calculate:

        20 ÷ (2 + 3)      .Ans : 20/ (5) ;  4.

6. Revenue Question

Price = ₹200

Quantity = 15

Find Revenue.       

 Ans: Price * Quantity  ;  ₹3000.

7. Profit Question

Revenue = ₹5000

Cost = ₹3200

Find Profit.    

Ans : Revenue - Cost ;   ₹1800 .

Answers

1. x = 7
